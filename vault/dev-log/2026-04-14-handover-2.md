## Known Issues / Next Session Priorities

### 1. Player movement appears to jump multiple tiles visually
The player moves exactly 1 grid cell per keypress (confirmed in code), but visually appears to jump more than one tile. Root cause NOT yet confirmed — hypotheses:
- Tile size too small relative to viewport (tiles ~19px on screen for 20x16 map)
- Camera zoom fits full map height, making individual tiles small
- Player sprite size doesn't scale with camera zoom
- Possible fix: camera should follow player (already has lerp code) instead of showing full map, so tiles appear larger and movement is relative to nearby tiles

### 2. Tile art is generic pixel patterns, not recognizable terrain
The tile classifier assigns tiles by brightness/edge/variance but the source ComfyUI art is one coherent illustration chopped into cells, not individually meaningful tiles. The classified tiles are visually distinct from each other (dark=obstacles, light=ground) but don't look like "grass" or "trees."

Future approaches from research:
- Per-tile ControlNet generation (most control, slowest)
- Template-guided img2img with canny edge reference
- Better ComfyUI prompting for grid-aligned individual tiles

### 3. Palette colors need tuning
The `deep-forest-16.hex` palette produces garish oranges/purples for a forest. Need a more natural forest palette.

### 4. Dungeon and town biomes not yet through full pipeline
Only field biome has been preprocessed + classified + PCK built. Dungeon and town have raw ComfyUI output but need `tileset:preprocess` + `tileset:classify` + PCK builds.

## Key Architecture Decisions
- IntGrid values map directly to atlas positions (value N → atlas col N%16, row N/16) when auto-layer rules aren't configured
- Collision is decoupled from visuals (separate LDtk layer)
- Both eras share same collision grid (can differ later via Terrain_Father/Son overlays)
- PCK provides art assets, LDtk provides layout — independent pipelines
- Web builds fetch PCK via HTTP from game server, save to user://, then load

## Commands Reference
```bash
task comfyui:tileset -- --biome=field    # Generate raw sheet
task tileset:preprocess -- field          # Grayscale + 16-level requantize
task tileset:classify -- field            # Rearrange to semantic order
task pck:build -- whispering-woods        # Build PCK
task build                                # Web export
task test                                 # All test suites
```

## Known Issues

### Mastra `onsessioninitialized` override (`@mastra/mcp` v1.4.2)

Mastra's `startHTTP()` hardcodes its own `onsessioninitialized` after spreading user options — last key wins in JS object literals. Our callback is silently dropped. **Workaround:** eager default queue at startup. **Documented in:** `vault/specs/2026-04-13-mcp-transport-design.md` under Known Issues. **Action:** file upstream issue on [mastra-inc/mastra](https://github.com/mastra-inc/mastra).

### Other (from prior session, still applicable)

- **SurrealDB SDK v2 vs v3** — `db.create(Table)` had CBOR encoding issues. Recorder uses `db.query("CREATE ...")` as workaround.
- **fastembed exit crash** — `libc++abi` crash after `process.exit(0)`. Cosmetic only.

Hold button or press Tab key to reveal world map

Normal Mode

  • Both game windows visible, map peeks around edges
  • Active window has gold border, full opacity
  • Inactive window has red border, slightly dimmed
  • Press a key to toggle which window is active

Map Peek Mode (hold Tab)

  • Windows slide off-screen to reveal full world map
  • Pulsing markers show each character's location
  • Labels identify which era is where
  • Release Tab to snap windows back into place
